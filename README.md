# Foodies For Foodies

A modern landing page for "Foodies For Foodies" - a vibrant community platform for food enthusiasts.

## Project Overview

This is a responsive single-page website showcasing the Foodies For Foodies platform with features including:

- **Navigation Bar** - Links to key sections and sign-in button
- **Hero Section** - Eye-catching introduction with featured category
- **Features Grid** - Showcase of 8 key platform features
- **Call-to-Action Section** - Conversion-focused signup prompt

1. Open `index.html` in your web browser

## Technical Stack

- **HTML5** - Semantic markup
- **CSS3** - Responsive design with CSS Grid and Flexbox
- **JavaScript** - Smooth scrolling and scroll animations

## Responsive Design

The website is fully responsive with breakpoints at 768px for mobile optimization.

## Browser Support

Works on all modern browsers that support:
- CSS Grid and Flexbox
- CSS3 animations
- ES6 JavaScript (IntersectionObserver API)
